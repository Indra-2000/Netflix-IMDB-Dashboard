#  Netflix IMDb Dashboard – Power BI

##  Project Overview

This Power BI dashboard provides an interactive analysis of Netflix titles using IMDb data. It helps users explore movie and TV show trends, ratings, genres, and content distribution through dynamic visualizations and KPIs.

The dashboard is designed to uncover insights into Netflix's content library by analyzing IMDb ratings, genres, and title performance.

---

##  Objectives

* Analyze Netflix content using IMDb data.
* Identify top-rated movies and TV shows.
* Explore genre-wise content distribution.
* Understand rating trends and content popularity.
* Create interactive visualizations for data-driven insights.

---

## Dashboard Features

*  IMDb Rating Analysis
*  Genre Distribution Analysis
*  Movies vs TV Shows Comparison
*  Content Trends and Performance Metrics
*  Interactive Filters and Slicers
*  Dynamic Charts and Visualizations

---

## Tools & Technologies

* **Power BI**
* **Power Query**
* **DAX (Data Analysis Expressions)**
* **Data Modeling**
* **Data Visualization**

---

## Key Insights

* Distribution of content across different genres.
* Identification of highly-rated Netflix titles.
* Comparison between movies and TV shows.
* Trends in content ratings and popularity.
* Genre preferences based on IMDb scores.
